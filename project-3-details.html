Gigboard

For this project I had the honor to be in a group of highly capable and intelligent individuals to develop this application we all called Gigboard. We all had a strong role in this project, where I dedicated most of my time to the backend with a couple of collegues while others were strongly inclined on the frontend. From API calls from the backend to the frontend, to using react-google-login to incorporate Google Oauth, this project taught us a lot. The usage of Github was our biggest obstacle to overcome but we managed to solve a few conflicts as a team and push forward. React was used as the frontend and we used MongoDB as the database. JavaScript was used throughout this whole project and it was finally deployed through Heroku.
